# NexJob Marketing Site

Public-facing marketing website for NexJob — the AI-powered field service management platform.

## Stack
- **Vite** + **React 18** + **Tailwind CSS 3.4**
- React Router v6 (SPA with 5 pages)
- lucide-react icons

## Getting Started

```bash
git clone https://github.com/sguilkey6-crypto/nexjob-marketing.git
cd nexjob-marketing
npm install
npm run dev
```

Opens at `http://localhost:3000`

## Pages
- `/` — Home (hero, features grid, AI demo, testimonials)
- `/features` — Feature deep-dives with interactive visuals
- `/pricing` — 4 tiers + Jobber comparison + savings calculator
- `/industries` — 9 trade-specific cards
- `/about` — Brand story + contact form
